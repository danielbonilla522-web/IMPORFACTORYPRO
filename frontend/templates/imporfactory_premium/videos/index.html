⚠ YouTube no configurado

Para usar este módulo, primero configura las credenciales OAuth en Google Cloud Console:

  1. Crea un proyecto en console.cloud.google.com
  2. Activa la API YouTube Data API v3
  3. Crea credenciales OAuth 2.0 Client ID (tipo Web application)
  4. Agrega como redirect URI: https://impor.imporchina.com/api/imporfactory/youtube/5/oauth/callback
  5. Guarda YOUTUBE_OAUTH_CLIENT_ID y YOUTUBE_OAUTH_CLIENT_SECRET en /configuracion
  6. Vuelve aquí y haz click en Conectar YouTube
